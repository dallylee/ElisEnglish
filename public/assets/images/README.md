# Images

This folder contains images for vocabulary words, characters, and theme decorations.

## File Naming Convention

Name image files to match their usage:
- Vocabulary images: `word-id.png` (e.g., `hello.png`, `goodbye.png`)
- Character avatars: `character-name.png` (e.g., `luna.png`, `hermione.png`)
- Theme images: Use descriptive names

## Image Formats

Supported formats:
- PNG (recommended for graphics and illustrations)
- JPG (recommended for photos)
- SVG (for icons and scalable graphics)

## Image Guidelines

### Vocabulary Images
- Size: 300x300px to 500x500px
- Style: Clear, simple, child-friendly illustrations
- Background: Transparent PNG preferred
- Content: Should clearly represent the word

### Character Avatars
- Size: 200x200px minimum
- Shape: Square or circular
- Style: Friendly, appropriate for age 10

## Sourcing Images

You can get images from:

1. **Free stock photo sites**:
   - Unsplash.com
   - Pexels.com
   - Pixabay.com

2. **Illustration sites**:
   - Undraw.co (SVG illustrations)
   - Freepik.com (requires attribution)

3. **AI image generation**:
   - DALL-E
   - Midjourney
   - Stable Diffusion

4. **Create your own**: Use drawing apps or commission an artist
